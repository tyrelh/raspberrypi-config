# NVM: Node Version Manager

## Installation
Go to [https://github.com/nvm-sh/nvm/blob/master/README.md#install--update-script](https://github.com/nvm-sh/nvm/blob/master/README.md#install--update-script) and follow instructions to install

## Use
To install the latest version of node:
```bash
nvm install node
```

To install a specific version:
```bash
nvm install 14.18.2
```

To list versions:
```bash
nvm ls
```

To switch version for the current shell:
```bash
nvm use 14.18.2
# or
nvm use node # uses the most current version
```

To set default node verson for new shells:
```bash
nvm alias defalt 14.18.2
```

For more info, see git page linked above.
