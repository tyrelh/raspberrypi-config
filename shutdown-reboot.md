# Shutdown or Reboot rpi
Taken from [https://raspi.tv/2012/how-to-safely-shutdown-or-reboot-your-raspberry-pi](https://raspi.tv/2012/how-to-safely-shutdown-or-reboot-your-raspberry-pi)

## Shutdown
```bash
sudo shutdown -h now  # h is hault

# or

sudo shutdown -h 5  # shutdown in 5 minutes
```

## Reboot
```bash
sudo shutdown -r now  # -r for reboot

# or

sudo reboot
```
